

# Maintenance Predictive for Wind Turbines

## Context

This project is part of a Final Year Project (PFE) in Data Science applied to industrial maintenance.

It is based on real SCADA data from wind turbine monitoring systems, with the objective of reducing costs related to unplanned failures.

## Objectives

- Anomaly Detection :	Identify abnormal behavior in real time
- Predictive Maintenance	: Anticipate failures 2 hours in advance
- Business Impact :	Quantify cost savings generated by the model Data

Three complementary datasets from a SCADA system:

- **SCADA**:	`scada_data.csv` => Real-time physical measurements (wind speed, power, temperatures, etc.)
- **Status**:	`status_data.csv` => Operational state of the turbine (running, stopped, maintenance, etc.)
- **Fault**:	`fault_data.csv`=> Recorded alarms and detected failures
- **Final**:	`df_final.csv`	Final dataset with labeled target built from the three sources

## Installation
- **Clone repository**
git clone https://github.com/NouhailaNR/maintenance-predictive.git 

cd maintenance-predictive

**Install dependencies**
```bash
pip install -r requirements.txt
```

**Run notebooks in order**
```bash
jupyter notebook notebooks/01_EDA_preparation.ipynb
jupyter notebook notebooks/02_anomaly_detection.ipynb
jupyter notebook notebooks/03_predictive_maintenance.ipynb
jupyter notebook notebooks/04_business_impact.ipynb
```
